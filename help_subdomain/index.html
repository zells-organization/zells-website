FAQ
To create a website on Tilda, you don't need to have any special skills. If you have a question, most likely you will find the answer below.
How to connect a custom domain?
Briefly: to connect a domain, add an A-record that points to 185.203.72.17. If you want to assign the WWW subdomain, add another A-record pointing to 185.203.72.17. You need to do this at your registrar (where you purchased the domain)
Learn more
How to duplicate a page or create another one?
To copy a page, go to Page Settings → Actions → Duplicate. To create more pages, go to Page List and click "Create a new page"
Learn more
I don't see changes after publishing
Browsers may store the old version of the page. If you don't see changes after publishing, please refresh the page.
How to remove the "Made on Tilda" label in the footer?
If you have purchased the annual Personal or Business plan, you can remove the "Made on Tilda" label in Site Settings. The label is placed on all pages by default.
How to add a link to a specific part of a page?
Add a block called "Anchor" (Blocks Library → Other → Anchor Link). Then enter the anchor name (for instance, "chapter1") in the block Contents panel. To point to this anchor, use the hashtag before the anchor name in the URL input field: "#chapter1".
Learn more
How to add a favicon (browser icon)?
To add a favicon, go to Site Settings → More → Favicon.
How to assign your own font?
To change the default font for the website, go to Site Settings → Fonts and Colors → Custom Settings
Learn more
How do I upgrade from my Personal to a Business plan?
To upgrade your plan, purchase the Business plan subscription. Unused days of your previous subscription will be accumulated and recalculated: 2 days of the Personal plan = 1 day of the Business plan.

What's the difference between a project and
a page?

A project is your future website. It may consist of several pages (a portfolio, magazine, or a blog), or of just one page (a landing page or a longform article).

The difference is in the settings: styles (e.g. fonts, colors) and domains are assigned to the entire project. Thus, if you want to create several interactive stories using one style (fonts and colors) publish them to the same domain, one project would be enough. If instead, you want to build several independent pages with different styles and publish them to different domains (e.g. landing pages for different customers), then you need to create several projects.
